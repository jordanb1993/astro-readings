# Friends' Natal Charts
*One file per person. Add birth data when available to unlock synastry readings.*

---

## What's Needed for a Full Chart

- Full name (or preferred name)
- Date of birth (month/day/year)
- Time of birth (as exact as possible — even "around 3pm" helps narrow the Ascendant)
- Place of birth (city/country)

With just date + place (no time): Sun, Moon, and all planets except Ascendant/Midheaven will be accurate.
With full birth data: complete chart including house placements.
