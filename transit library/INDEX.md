## How to Use This Index

- **At session start:** Scan the orb column. Anything at 0–2° applying = current headline.
- **For daily readings:** Long-term transits reference here by link, not re-explained in full.
- **To add a new transit:** Create `transit-library/[planet]-[aspect]-[natal-point].md` using `_template.md`, then add a row here.
- **To archive:** When a transit leaves 5° orb and won't retrograde back, move it to "Recently Completed." Delete from index when orb exceeds 8°.
- **Update orbs:** At each session, update the orb column using live-fetched planet positions. Don't let the index go stale.
