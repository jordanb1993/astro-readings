# ritual/

Moon rituals, tarot pulls tied to specific events, and other ceremonial readings.

**Not in `readings/`** — these are intention-setting or reflective documents, not transit readings.

## What lives here
- New moon / full moon ritual notes
- Tarot pulls with reflective write-ups (not casual daily pulls — those go in `tarot-log.md`)
- Any reading done for a specific ceremonial or spiritual purpose

## Naming convention
`YYYY-MM-DD-[descriptor].md`

Examples:
- `2026-05-16-new-moon-ritual.md`
- `2026-05-14-tarot-eleanor.md`
