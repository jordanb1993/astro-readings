## Aspect Calculation Rules — CRITICAL

**Step 1 — Absolute degree formula:**
```
absolute_degree = (sign_number × 30) + degree_decimal
```
Where sign numbers are: Aries=0, Taurus=1, Gemini=2, Cancer=3, Leo=4, Virgo=5, Libra=6, Scorpio=7, Sagittarius=8, Capricorn=9, Aquarius=10, Pisces=11

**Examples:**
- Moon at 4°58' Aries → (0 × 30) + 4.97 = **4.97°**
- Sun at 1°49' Sagittarius → (8 × 30) + 1.82 = **241.82°**
- Pluto at 25°40' Scorpio → (7 × 30) + 25.67 = **235.67°**

**Step 2 — Angular difference:**
```
diff = abs(planet1_abs - planet2_abs)
if diff > 180: diff = 360 - diff
```

**Step 3 — Aspect check:**
| Aspect | Angle | Orb (natal-to-transit) |
|--------|-------|----------------------|
| ☌ Conjunction | 0° | ±6° |
| ☍ Opposition | 180° | ±6° |
| □ Square | 90° | ±6° |
| △ Trine | 120° | ±6° |
| ✶ Sextile | 60° | ±4° |
| ⚻ Quincunx | 150° | ±2° |

**Applying vs. separating:**
- **Applying:** transiting planet moving TOWARD exact aspect — strengthen reading
- **Separating:** transiting planet moving AWAY from exact — note it happened, use past tense
- Separating >5°: remove from signal board (stale transit discipline — added June 8, 2026)
- Calculate daily velocity to determine: Venus ~1.2°/day, Jupiter ~0.083°/day, Saturn ~0.034°/day, Neptune ~0.017°/day

**Stale transit rule (June 8, 2026):**

import math

def sign_to_abs(sign_name, degrees, minutes=0):
    signs = ['Aries', 'Taurus', 'Gemini', 'Cancer', 'Leo', 'Virgo',
             'Libra', 'Scorpio', 'Sagittarius', 'Capricorn', 'Aquarius', 'Pisces']
    sign_num = signs.index(sign_name)
    return (sign_num * 30) + degrees + (minutes / 60)

def angular_diff(a, b):
    diff = abs(a - b)
    if diff > 180:
        diff = 360 - diff
    return diff

def check_aspect(diff, orb=6):
    aspects = {
        'conjunction': (0, orb),
        'sextile': (60, 4),
        'square': (90, orb),
        'trine': (120, orb),
        'quincunx': (150, 2),

    }
    for name, (angle, allowed_orb) in aspects.items():
        if abs(diff - angle) <= allowed_orb:

    exact_diff_tomorrow = abs(diff_tomorrow - aspect_angle)
    return exact_diff_tomorrow < exact_diff_today
