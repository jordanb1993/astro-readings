# Readings Archive
*Dated transit readings — one file per session*

---

## Folder Structure

```
readings/
├── YYYY-MM-DD.md           ← current month's daily readings
├── 2026-04/                ← completed months, archived after they close
│   ├── 2026-04-21.md
│   └── ...
├── synastry/               ← Jordan's chart vs. another person
│   └── YYYY-MM-DD-[name]-synastry.md
├── workplace/              ← weekly workplace transit digests
│   └── YYYY-MM-DD-week.md
└── _template.html          ← legacy HTML template (reference only)
```

**Moon rituals and ceremonial tarot pulls** live in `ritual/` at the workspace root — not here.

---

## File Naming

`YYYY-MM-DD.md` — e.g., `2026-05-03.md`

For multiple readings on the same day: `2026-05-09-evening.md`

---

## Archiving

When a full calendar month has passed, move all files for that month into `readings/YYYY-MM/`. Current month always stays at root level of `readings/`.

---

## What to Include

- Transiting planets and degrees used
- Full reading text
- Any follow-up reflection Jordan added after the fact
- Verified correlations to flag for KNOWLEDGE.md
